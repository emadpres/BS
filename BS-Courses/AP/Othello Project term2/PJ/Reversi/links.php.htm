Othello programs

Edax (6590)
Richard Delorme's very strong program. The 3.2 evaluator is stronger than ever. (Text based interface)

Gunnar's Othello Page (Wzebra) (557)
The best all-around program for the average user, bar none, provided free courtesy of Gunnar Andersson.

Chris' Othello Stuff (7507)
Quite likely the strongest program in the world-certainly it is the strongest one publically available. Not nearly as user friendly as Wzebra, though; but Edmund Yiu's NBrowser helps that a little bit. Provides very good features such as draw avoidance.

Happy End (209)
Beppi Mennozi's sophisticated endgame solitaire program.

Booby Reversi (343)
Directly playable on the site at its lower levels, full version downloadable.

Icare (328)
A small but sweet endgame solitaire program.

Cassio (265)
Stephane Nicolet's program, the Mac equivalent of Wzebra.

Cyrano (340)
Worthy Othello applet. Has a beautiful entrance graphic.

Devastator (5497)
One of the strongest Othello applets online.

Darwersi (189)
Darwinian Reversi- Reversi by genetic algorithm.

Andrea Zinno's Home Page (136)
Andrea Zinno's program. The website has other links and resources, too.

http://www.tournavitis.de/herakles/ (256)
This a great suite of Othello programs- two strong 8x8 programs, and a strong Diamond and 10x10 program.

Virtual Othello (237)
A Russian program, that also plays on GGS as vo

Pointy Stone (161)
A Windows based program

Logistello (387)
The breakthrough program that completely dominated all humans and programs who came before it.

Thor (309)
The files of the primary Othello game database

OthBase (157)
Fabrice Di Meglio's Java database program. Works on many platforms, enables database creation, updates, browsing and searching. Shows statistics about games. Reads XOF, Thor, VOG, Kurnik, GGS games files.

Game Sites

GGS (250)
This site where top programs play against each other, and a few strong humans as well. We recommend using GGS through Lion.

Kurnik (156)
Originally a Polish game site, it is the highest quality free site for Othello that I know of. Has game archives and regular tournaments. The site can be viewed in English. Uses Java applets.

VOG (66)
Regular tournaments, variants, archives, many other features. Free trials for tournaments, but $29.95 per annum for full use of the site.

Yahoo USA (87)
(Need to log into a yahoo account first) Uses Java applet. The starting position is inverted, but other than that it's very nice.

Nethello (128)
(In Japanese) Quality Japanese site.

Blip (77)
(In Swedish) A Swedish game site run by the television channel 4. Some top Nordic players play here. Go to Spela -> Bradspel -> Othello.

MSN Gaming Zone (99)
Popular gaming site.

Spelpunt (45)
(In Dutch) The Dutch site.

Thai Board Games Online (59)
(In Thai)

Chinagames.net (55)
(In Chinese) A Chinese game site for Othello. Has a few weird features but otherwise alright.

Little Golem (36)

It's Your Turn (39)
